Simpler Sidebar

A simple side nav in jQuery

Dynamic sidebar loaded with ajax

Press (the menu icon) or (the more icon) to trigger the sidebar - but also to navigate this demo.

Welcome!

This playground is delivered with the simpler-sidebar plugin, you can find it in the demo directory. It also relies on the jquery library via CDNs. Turn on internet to make it work.

If you need a guide to install the plugin, you are in the wrong place. Check the Getting Started guide in the main website or directly on the simpler-sidebar github page at this link.
